    Ohms lag #

    Ohms lag är den mest grundläggande inom ellära och även den viktigaste. Denna formel används för att kunna räkna ut spänningar, strömmar och resistanser i olika enkla kopplingar.

    Genom algebra så kan formeln göras om så man kan räkna ut resistansen och strömmen. Alla tre varianterna av Ohm´s lag ses nedan.
